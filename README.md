# 🔐 User Registration System – Django Web App

## 📌 Overview
This is a Django-based User Registration System that allows users to create an account, log in, and manage authentication securely. It demonstrates the core concepts of user authentication in web applications.

---

## 🚀 Features
- 📝 User Registration (Sign Up)
- 🔑 User Login
- 🚪 User Logout
- 🔒 Authentication system using Django
- 🎨 Responsive UI with Bootstrap

---

## 🛠️ Tech Stack
- Python 🐍
- Django 🌐
- HTML, CSS, Bootstrap 🎨
